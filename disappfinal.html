I worked on a restaurant application project with a team of 11 people. I was the UI specialist so I was in charge of the look and feel of the application.

After many trial and errors, I finally decided to go for a cafe feel for our application and make a mock restaurant application for a cafe called "Cafe Maji".

I did so by having a nice soft and earthy color scheme. I first drew out drafts of the layout and then created all the necessary buttons and backgrounds for it in Adobe Photoshop CS5. Fonts and colors were hand picked by yours truly and all of this was inserted into our android application via XML and Java.

However I do still have some residual regret about the final UI look. If given the time, I would continue to make changes to the overall UI of DISAPP.

Type: User Interface, Android Application
Tools: Adobe Photoshop CS5, Java, XML
